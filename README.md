<h2><img width="20px" src="https://github.com/christianebs/christianebs/assets/108686840/45469bba-2011-45d3-8d45-d03d56753e32"> Sobre o Projeto </h2>

Este projeto é um desafio para desenvolver um site, do zero, com personagens do X-Men. Foram 3 aulas ao vivo sobre HTML, CSS e JavaScript e 3 vídeos sobre como iniciar na carreira de programação.
